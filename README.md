# Micha Stocks Fan — Next.js App

A web portal for the Micha Stocks Course that displays lessons, transcripts, and course content using a modern UI with infographics and video playback.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Linting:** ESLint

## Project Structure

```
src/
├── app/
│   ├── layout.tsx        # Root layout
│   ├── page.tsx          # Home / course index page
│   ├── globals.css       # Global styles
│   └── lesson/           # Dynamic lesson pages
├── components/
│   ├── ConceptCard.tsx   # Card component for concept display
│   ├── CourseBrowser.tsx # Course browser component
│   ├── Footer.tsx        # Site footer
│   ├── InlineMarkdown.tsx# Inline markdown renderer
│   └── VideoPlayer.tsx   # Embedded YouTube video player
└── data/
    ├── course.json       # Generated course data (do not edit manually)
    ├── sections.json     # Generated section data (do not edit manually)
    └── sections.ts       # Section data runtime interface
```

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Building for Production

```bash
npm run build
npm start
```

## Updating Course Data

Course data files (`src/data/course.json` and `src/data/sections.json`) are auto-generated by the pipeline in the [micha-stocks-fan](https://github.com/Ariel-B/micha-stocks-fan) repository. Do not edit them manually.
