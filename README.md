# Micha Stocks Fan — Next.js App

A web portal for the Micha Stocks Course that displays lessons, transcripts, and course content using a modern UI with infographics and video playback.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Linting:** ESLint
- **Key dependencies:** `framer-motion`, `lucide-react`, `react-markdown`, `recharts`, `clsx`, `tailwind-merge`

## Project Structure

```
src/
├── app/
│   ├── layout.tsx        # Root layout
│   ├── page.tsx          # Home / course index page
│   ├── globals.css       # Global styles
│   ├── lesson/           # Dynamic lesson pages
│   └── reports/          # Market insight reports browser + detail pages
├── components/
│   ├── ConceptCard.tsx           # Card component for concept display
│   ├── CourseBrowser.tsx         # Course browser component
│   ├── Footer.tsx                # Site footer
│   ├── InlineMarkdown.tsx        # Inline markdown renderer
│   ├── PrintableReportContent.tsx# Print-friendly report layout
│   ├── ReportCard.tsx            # Summary card for a single report
│   ├── ReportMarkdown.tsx        # Markdown renderer for report content
│   ├── TickerLinks.tsx           # Converts ticker symbols to external links
│   └── VideoPlayer.tsx           # Embedded YouTube video player
├── data/
│   ├── course.json       # Generated course data (do not edit manually)
│   ├── sections.json     # Generated section data (do not edit manually)
│   └── sections.ts       # Runtime interface — exports SECTIONS, LESSON_SECTION, SECTION_COLORS, getSectionForLesson, findSection, getLessonNum
└── lib/
    └── reports.ts        # Server-side helpers for reading report files from content/reports/
```

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Building for Production

```bash
npm run build   # Compile & generate static pages
npm start       # Serve the production build
```

## Linting

```bash
npm run lint
```

## Updating Course Data

Course data files (`src/data/course.json` and `src/data/sections.json`) are auto-generated by a separate data pipeline. Do not edit them manually.

## Data Types

Key types used across the app:
- `Lesson.takeaways` — `{ text: string; level: number }[]` (not a plain string array)
- `Lesson.concepts` — `{ title: string; body: string; timestamp?: number }[]`
- `SectionDef` — `{ id: string; label: string; color: string }` (from `sections.ts`)
